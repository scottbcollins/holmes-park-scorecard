# Holmes Park — Golf Betting Scorecard

A mobile-friendly golf betting scorecard for the format:
- **Low Total** — 2 points per hole (team combined strokes)
- **Low Ball** — 1 point per hole (best ball)
- **Bonus** — Birdie +1, Eagle +3

Teams: **Ben & Tucker** vs **Tyler & Scott**

## Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2 — GitHub + Vercel Dashboard
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Click Deploy — done

No build step needed. It's a single static HTML file.
